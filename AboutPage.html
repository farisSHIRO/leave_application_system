About Rexaro


Scope

The aim of this project is to develop an online leave application system that is necessary for a firm like Rexaro. The Employee Leave Application System is intended to automate employee administration and management, as well as strategic leave planning for employees. This system also keeps track of all personnel records. At the same time, this system also provides a platform that will automate employee leave administration in a way that is easy for both the admin and employees. Our system can also handle all stages of leave management, such as leave application, approval, and rejection.

Objective


The main objective of this project is to reduce paperwork and to help maintain the database. There are several other objectives that we as a group have encountered in this project such as:

1) For the employees to be able to request for their leave in the leave request form.

2) For the employees to be able to view their check status.

3) For the administrator to be able to perform the approval of the leave request.

4) For the administrator to be able to check the status.

INTRODUCTION

Leave Application System

We as group came to a decision to choose the leave application system as our project title. Employees must take time off to relax and recoup in order to prevent being overworked. As a result, our system assists the company called REXARO in managing their employee time off requests in a more efficient, fair, and accurate manner. In order to access our leave application system, the user first must log in as an administrator or employee. If the user selects to login as an employee, they must provide an Employer Name and Employer Password. The employee can obtain a leave of absence and complete out the necessary paperwork. When seeking leave, the employee must keep in mind certain restrictions, such as 8 days per year for employees who have worked for 1-2 years. Also, if you have been with the company for 1-2 years, you may be eligible for 14 days of medical leave every year. Finally, a casual leave policy allows for a maximum of 8-15 days of casual leave each year. The employee can use the check status page to see the status of their leave application.
